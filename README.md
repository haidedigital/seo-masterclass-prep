## Какво има в репото

### [`masterclass-prep/masterclass-prep-50-point-checklist.md`](masterclass-prep/masterclass-prep-50-point-checklist.md)
60-точков GEO технически чеклист за 2026 плюс пълен скрипт на лекцията в три акта:

- **Акт 1 — Как всъщност ви вижда Google** (15 мин): 2MB HTML лимитът, Web Rendering Service statelessness, per-resource crawl budget, мартенския 2026 Core Update и Information Gain сигнала.

Чеклистът е фактологично проверен спрямо първоизточници: Google Search Central, OpenAI, Anthropic, Perplexity, Mistral, Apple, Common Crawl, IETF AI Preferences draft, HTTP Archive, iPullRank и Search Engine Land.
